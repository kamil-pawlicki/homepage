Notatki z WTF: co ten frontend

Tydzień 1

Wprowadzenie, wyjaśnienie czym jest frontend, instalacja Visula Studio Code, prezentacja ścieżki rozwoju, wprowadzanie zmian na gotowej stronie.

Tydzień 2

Wytłumaczenie zasady działania warstw frontendu. Html wpływa na treść. CSS odpowiadający za style wpływa na HTML oraz na Treść. Java odpowiedzialna za funkcje i klasy wpływa na CSS, HTML i Treść.
Podstawowe polecenia języka HTML, takie jak: składnia <<>, h1-h6, section, article itp.
Podstawowe polecenia języka CSS, takie jak: składnia {}, selektor {wartość:"";}, color, background-color, link href="nazwa pliku" rel="stylesheet" type="text/css".
Porównanie programowania do gotowania. Schemat blokowy.
W kodowaniu jak w gotowaniu - Potrzebne jest: spis składników potrzebnych do przygotowania potrawy; dokładnie opisany przepis; prawidłowo wykonane czynności.
Poznanie DevTools komendą F12 na witrynie internetowej. sprawdzanie poszczególnych elementów Prawym klawiszem Myszy "zbadaj" . Możliwe są edycje strony w celu podglądu wersji. Po odświeżeniu wszystko wraca do normy.
Zadania domowe. Stworzyć Homepage, Index, style.css, notatki z WTF,
